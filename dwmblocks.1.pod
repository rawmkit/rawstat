=head1 NAME

dwmblocks - modular status bar for dwm

=head1 SYNOPSIS

B<dwmblocks> [B<-d> I<DELIM>] [B<-p>]

=head1 DESCRIPTION

dwmblocks is a modular status bar for dwm written in C.  The status
bar is made from text output from command-line programs.  Blocks are
added and removed by editing F<config.h> file.

=head1 OPTIONS

=over 8

=item B<-d> I<DELIM>

Set blocks delimeter.

=item B<-p>

Print status bar to stdout.

=back

=head1 CUSTOMIZATION

B<dwmblocks> can be customized by creating a custom F<config.h> and
(re)compiling the source code.  This keeps it fast, secure and simple.

=head1 SEE ALSO

L<dwm(1)>, L<dmenu(1)>, L<st(1)>
